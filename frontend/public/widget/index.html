Revix Widget · Demo

Página de prueba del widget embeddable. Pulsa el botón azul de la esquina inferior derecha para abrir el chat.

INSTALACIÓN

Una sola línea

<script src="https://revix.es/widget/revix-widget.js" defer></script>
OPCIONES

Personalización (opcional)

<script
  src="https://revix.es/widget/revix-widget.js"
  data-color="#1d4ed8"
  data-position="right"
  data-title="Asistente Revix"
  data-greeting="Hola, ¿qué dispositivo quieres reparar?"
  defer></script>
API

Control programático

RevixWidget.open();   // abrir
RevixWidget.close();  // cerrar
RevixWidget.version;  // "1.0.0"
