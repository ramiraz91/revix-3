1. Introducción

¿Qué es NEXORA?

NEXORA es un sistema CRM/ERP diseñado específicamente para talleres de reparación de dispositivos móviles. Permite gestionar de forma integral todas las operaciones del negocio.

🔄 Automatización

Polling automático de Insurama, notificaciones en tiempo real, sincronización de estados.

🤖 Asistente IA

Diagnósticos y sugerencias de reparación con inteligencia artificial Gemini.

📱 Multi-dispositivo

Acceso desde ordenador, tablet o móvil con interfaz adaptable.

👥 Multi-usuario

Roles diferenciados: Master, Admin y Técnico con permisos específicos.

4. Órdenes de Trabajo

Estados de una Orden

Pendiente Recibir → En Diagnóstico → Presupuesto Enviado → En Reparación → Reparado → Entregado

Crear Nueva Orden

  1. Pulsar "+ Nueva Orden"
  2. Buscar o crear cliente
  3. Rellenar datos del dispositivo (marca, modelo, IMEI)
  4. Describir el problema
  5. Seleccionar tipo de servicio
  6. Pulsar "Crear Orden"

5. Gestión de Clientes

Lista de Clientes

Muestra todos los clientes registrados con su información de contacto y historial.

CampoDescripción

DNI/NIFDocumento de identificación
TeléfonoPara notificaciones SMS
EmailPara notificaciones por correo

Crear Nuevo Cliente

  1. Pulsar "+ Nuevo Cliente"
  2. Rellenar nombre y apellidos
  3. Introducir DNI/NIF
  4. Añadir teléfono (para SMS)
  5. Añadir email (para notificaciones)
  6. Completar dirección
  7. Pulsar "Guardar"

Ficha de Cliente

Al abrir un cliente se muestra:

  • Datos personales: Toda la información de contacto
  • Historial de órdenes: Todas sus reparaciones anteriores
  • Métricas: Total gastado, número de órdenes completadas
  • Notas: Observaciones especiales sobre el cliente
💡 Tip: Al crear una orden, puedes buscar clientes existentes por nombre, teléfono o DNI.

6. Inventario

Gestión de Repuestos

El inventario permite controlar todo el stock de repuestos del taller.

Añadir Nuevo Repuesto

  1. Pulsar "+ Nuevo Repuesto"
  2. Introducir nombre del producto
  3. Seleccionar categoría
  4. El SKU se genera automáticamente
  5. Añadir precios y stock inicial
  6. Pulsar "Guardar"
📝 SKU Automático: El código SKU se genera basándose en la categoría y nombre del producto. Ejemplo: PANT-IPHO-PRO-A1B2

Categorías de Repuestos

  • Pantallas: LCDs, OLEDs, táctiles
  • Baterías: Originales y compatibles
  • Conectores: Puertos de carga, flex
  • Cámaras: Frontales y traseras
  • Altavoces: Auriculares y altavoces
  • Carcasas: Tapas traseras, marcos
⚠️ Alerta de Stock: El sistema notifica automáticamente cuando el stock de un producto baja del mínimo configurado.

7. Integración Insurama/Sumbroker

¿Qué es Insurama?

Insurama es el portal de gestión de siniestros de seguros de móviles. NEXORA se integra directamente para automatizar el proceso.

✅ Funcionalidades:
  • Recibir automáticamente nuevos siniestros
  • Enviar presupuestos completos al portal
  • Sincronizar estados de reparación
  • Gestionar fotos y evidencias

Enviar Presupuesto a Insurama

Campos obligatorios del formulario:

CampoDescripción
Precio (€)Coste total de la reparación
DisponibilidadInmediata, 24h, 48h, 7 días, Sin stock
Tiempo en HorasHoras estimadas de trabajo
Tipo de RecambioOriginal, Compatible, Reacondicionado
Tipo de GarantíaFabricante, Taller, Sin garantía
DescripciónDetalle de la reparación a realizar

8. Pre-Registros

¿Qué son los Pre-Registros?

Los Pre-Registros son siniestros de Insurama que aún no tienen orden de trabajo creada. Representan trabajos potenciales que el sistema ha detectado automáticamente.

Lista de Pre-Registros

Muestra todos los siniestros pendientes con:

CampoDescripción
Código de siniestroIdentificador único (26BE...)
ClienteNombre del asegurado
DispositivoModelo del móvil
EstadoPendiente, Enviado, Aceptado, etc.
FechaCuándo se recibió el siniestro

Acciones Disponibles

BotónAcción
Enviar PresupuestoIr a Insurama para enviar precio
Crear OrdenGenerar orden de trabajo directamente
Ver OrdenAbrir orden existente (si ya se creó)

Estados de Pre-Registro

  • Pendiente Presupuesto - Sin precio enviado
  • Presupuesto Enviado - Esperando respuesta del cliente
  • Aceptado - Cliente aprobó → Crear orden
  • Rechazado - Cliente no aceptó
  • Orden Creada - Ya tiene orden de trabajo
  • Cancelado - Siniestro anulado
🗑️ Limpieza Automática: Los pre-registros cancelados, rechazados y archivados se eliminan automáticamente después de 7 días.

Para gestionar varias notificaciones a la vez:

  3. Usar "Seleccionar todas" o "Deseleccionar"
  4. Pulsar "Eliminar (N)" o "Marcar leídas"

11. Flujos Automáticos

🔄 Polling de Insurama

¿Qué hace? Cada 5 minutos el sistema consulta automáticamente el portal de Sumbroker buscando nuevos siniestros y cambios de estado.
1. Sumbroker envía siniestro → 2. NEXORA detecta (polling) → 3. Se crea Pre-Registro → 4. Notificación al admin → 5. Envío de presupuesto → 6. Cliente acepta → 7. Orden creada automáticamente

12. Guía para Técnicos

Acceso al Panel de Técnico

  1. Iniciar sesión con credenciales de técnico
  2. Acceder a "Panel Técnico" en el menú
  3. Ver solo las órdenes asignadas a ti

✅ Validar Materiales Usados

Sistema de "picking" para confirmar que los materiales asignados realmente fueron utilizados:

  1. Ir a la pestaña "Materiales" de la orden
  2. Pulsar "Validar (N pendientes)"
  3. Escanear el SKU o código de barras de cada material
  4. El sistema confirma automáticamente al detectar el código
  5. Se registra quién validó y cuándo
Progreso visible:
  • Barra de progreso: X/Y materiales validados
  • Verde = Material validado
  • Gris = Pendiente de validar

Añadir Materiales

Desde Inventario:

  1. Pulsar "Existente"
  2. Buscar el repuesto por nombre o SKU
  3. Confirmar (la orden se bloquea hasta aprobación del admin)

Material No Registrado:

  1. Pulsar "Nuevo"
  2. Escribir nombre del material
  3. Indicar cantidad
  4. El admin asignará los precios
⚠️ Importante: Cuando un técnico añade material, la orden queda BLOQUEADA hasta que el administrador lo apruebe.

Subir Evidencias (Fotos)

  1. Ir a la pestaña "Evidencias"
  2. Pulsar "Subir Foto"
  3. Seleccionar imagen del dispositivo
  4. Añadir descripción
  5. La foto se guarda con la orden

Comunicación con Admin

  1. Ir a la pestaña "Mensajes"
  2. Escribir mensaje
  3. Pulsar "Enviar"
  4. El admin recibe notificación en tiempo real
